### 🔍 根因

`FeishuChannel` 有两个连接方法：

| 方法 | 适用场景 |
|------|---------|
| `connect()` | 同步脚本（`asyncio.run(channel.connect())`） |
| `start_background()` | **异步应用**（FastAPI/uvicorn） |

我们之前用了 `connect()` 包在 `asyncio.wait_for()` 里，uvicorn 事件循环嵌套冲突 → `"This event loop is already running"`

### 🛠️ 三个改动

| 文件 | 改动 |
|------|------|
| `bot.py` | `connect()` → `start_background(timeout=15)` |
| `routers/feishu.py` | `asyncio.wait_for(bot.connect())` → `asyncio.create_task(_connect_bot(bot))` 后台连接 |
| `main.py` | `bot.stop()` → `await bot.stop()` |
